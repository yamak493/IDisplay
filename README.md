# IDisplay

A simple Fabric mod that displays detailed item information when hovering over items in the inventory.

## Features

- **Item ID Visibility**: Shows the internal system ID, such as `minecraft:apple`.
- **Data Component Display**: Displays additional data components attached to the item.
- **Shift Key Integration**: Information is only shown while holding the **Shift key**.
- **Smart Visibility**: The component line is automatically hidden if the item has no components.

## How to Use

1. Install the mod and launch Minecraft.
2. Open your inventory.
3. Hover over an item and hold the **Shift key**.
   - The first line displays the Item ID.
   - The second line displays the component data (only if present).

## Environment

- **Minecraft**: 1.21.x (Fabric)
- **Java**: 21 or higher
